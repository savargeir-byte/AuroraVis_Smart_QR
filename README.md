# AuroraVision Smart QR Redirect

Smart QR code landing page that redirects users to the correct app store based on their device.

## URLs

- **Main redirect page**: https://savargeir-byte.github.io/AuroraVis_Smart_QR/
- **Android coming soon**: https://savargeir-byte.github.io/AuroraVis_Smart_QR/android-soon.html

## How it works

1. iOS users → App Store
2. Android users → Coming Soon page
3. Desktop/Other → Coming Soon page
